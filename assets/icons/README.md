# Icon Sources

Custom application icon SVG sources for YasserOS-specific applications.

## Files

- `yasser-control-center.svg` — Yasser Control Center application icon

## Deployment

Exported PNG icons are installed to:
```
/usr/share/icons/hicolor/{size}x{size}/apps/yasseros.png
/usr/share/pixmaps/yasseros.png
```

See `branding/icon-requirements.md` for size requirements and icon theme strategy.
