Raspberry Pi Setup

Supported Hardware

ModelArchitectureStatus
Raspberry Pi 4 Model Barm64 / aarch64Supported
Raspberry Pi 5arm64 / aarch64Supported
Raspberry Pi 3B+armv7 / arm64Community tested
Raspberry Pi Zero 2 Warm64Limited RAM — may be slow

Minimum Requirements

  • 2 GB RAM (4 GB recommended)
  • 16 GB SD card (32 GB recommended for Lab Mode)
  • Raspberry Pi 4 or later for best experience

Flashing

Use Raspberry Pi Imager (recommended) or flash manually:

xzcat yasseros-arm64.img.xz | sudo dd of=/dev/mmcblk0 bs=4M status=progress conv=fsync
sudo sync

Verifying with Pi Checker

After booting, open Yasser Control Center → Pi Checker to confirm:

  • ARM64 architecture detected
  • Raspberry Pi model identified
  • All required packages installed
