Quick Start

1. Flash the Image

  1. Download the latest yasseros-*.img.xz from the releases page.
  2. Flash with Raspberry Pi Imager or dd: 
    xzcat yasseros-*.img.xz | sudo dd of=/dev/sdX bs=4M status=progress conv=fsync
  3. Insert the SD card into your Raspberry Pi and power on.

2. First Boot

  1. YasserOS will display the splash screen during boot.
  2. Log in with the default credentials (set during the build or first-boot wizard).
  3. The XFCE desktop loads automatically.

3. Launch Control Center

Click the YasserOS menu button in the taskbar, or run:

ycc

4. Enable Lab Mode (optional)

From the Control Center, navigate to Lab Mode and toggle it on, or:

ycc --page lab

5. Update

yos-update

This runs sudo apt update && sudo apt upgrade -y.
